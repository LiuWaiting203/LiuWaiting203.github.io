# 用 JavaScript 做個簡單的井字遊戲！

之所以想要用 JavaScript 做簡單的井字遊戲，一方面是剛好適合我這個剛轉職前端工程師的菜鳥外
也很適合磨練自己的技術！而且也很喜歡玩井字遊戲（雖然是個很容易平局的遊戲 😅）
在此篇中，會用到以下框架和技術及工具

# 框架：

  • HTML5
  • CSS
  • JavaScript

# 工具：

  • CDNJS
  • Bootstrap 5 (方便快速修改、新增樣式)

# 使用到的技術：

  • LocalStorage (儲存勝利次數，載入勝利次數)
  • 提示玩家目前誰是 "X" 或 "O"
  • 若平局，按按鈕重新開始
  • 若勝利，會自動重新開始
  • 上傳到 Github 並用 Github Page 部屬

# JavaScript DOM Event 的運用！

那麼，我們要怎麼在九宮格內，點擊格中的空白處時，會新增 X 或 O 呢？
這時候我們就要用到 JavaScript 最有趣的部份， Event 事件！
使用 Event 等 DOM 事件去 監聽(Listening) 和 處理(Event Handling)

開始遊戲時，我們會用 EventListener(事件監聽) 去監聽按鈕和監聽按鈕後不能再重複按一次
用 Click Event(點擊事件) 觸發點擊格內空白處時，會新增 X 和 O 的欄位
在根據勝利的組合，並排查出誰贏了此局
監聽目前輪到哪個玩家扮演哪一方，在點擊 reBtn 按鈕時會重新開始布局
最後用 LocalStorage 去儲存勝利的次數，並在下次開啟網頁時，從 LocalStorage 中抓取資料並呈現在網頁上
